### 基础项目

#### 项目结构说明
- `backend/` - 后端代码目录（Spring Boot + MyBatis Plus）
  - `src/` - 源代码和资源文件
  - `database/` - 数据库脚本和SQLite数据库文件
  - `logs/` - 应用日志文件
  - `target/` - Maven构建输出
- `frontend/` - 前端代码目录（Vue 2 + Element UI）
  - `src/api/` - API接口封装
  - `src/views/` - 页面组件
  - `src/components/` - 公共组件
  - `src/router/` - 路由配置
  - `src/store/` - 状态管理
- `document/` - 项目文档

#### 后端技术栈
- Java 21
- Spring Boot 4.0.5
- MyBatis Plus
- MySQL / SQLite
- JWT认证

#### 前端技术栈
- Vue 2.7
- Vue Router 3
- Vuex 3
- Element UI 2.15
- Axios 1.6
- Vue CLI 5
